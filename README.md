# Rainfall Statistical Analysis

Analyses rainfall data from the Bureau of Meteorology and returns requested statistics of the given data. 

## Usage

```
rainStats rainfall.csv year/years stats/corr
```

Give the file a path to the rainfall data from the Bureau of Meteorology. 
If performing a statistical analysis of a single year, input the year, and 'stats'.
If performing a correlation analysis between two years, input the years as a list of two integers, and 'corr'.
